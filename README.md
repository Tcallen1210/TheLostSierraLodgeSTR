# The Lost Sierra Lodge STR Website

Static HTML/CSS/JS site for GitHub Pages.

## Where to edit
- Booking embed + external booking button: `pages/book.html`
- Site-wide style: `styles.css`
- Mobile menu + footer year: `script.js`
- Photos: replace files in `/assets`

## Publish (GitHub Pages)
Repo → Settings → Pages → Deploy from `main` / root.
